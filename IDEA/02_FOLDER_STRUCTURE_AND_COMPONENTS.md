# 📂 02. 공실뉴스 프론트엔드 폴더 구조 및 컴포넌트 설계안 

본 문서는 공실뉴스 포털 및 B2B 서브도메인의 Next.js (App Router) 프론트엔드 아키텍처 규칙을 명시합니다. 
스파게티 코드를 방지하고 유지보수성을 극대화하기 위해 코드를 '레고 블록(Component)'처럼 조각내어 관리함과 동시에, **'통합형 어드민(Unified Dashboard)'** 개념을 도입하여 코드를 압축합니다.

---

## 🏗️ 전체 폴더 트리 구조 (Directory Structure)

```text
src/
 ┣ 📂 app/                 # 1️⃣ 페이지(라우팅) 담당 폴더
 ┃ ┣ 📜 layout.tsx         # 모든 페이지에 공통으로 들어가는 HTML 최상위 뼈대
 ┃ ┣ 📜 page.tsx           # 메인 포털 루트 페이지 (공실뉴스 홈)
 ┃ ┣ 📂 news/              # 뉴스 카테고리별 페이지 및 기사 읽기 (/news)
 ┃ ┣ 📂 gongsil/           # 공실(매물) 리스트 및 지도 탐색 (/gongsil)
 ┃ ┣ 📂 study/             # 부동산특강 강의 리스트 및 수강 (/study)
 ┃ ┣ 📂 boards/            # 전사 공용 자료실 및 자유게시판 동적 라우팅 (/boards/[board_id])
 ┃ ┃
 ┃ ┣ 📂 admin/             # ⚙️ [통합 어드민 허브] 일반인/중개사/본사 공용 대시보드
 ┃ ┃ ┣ 📜 layout.tsx       # 공용 사이드바 뼈대 (Role에 따라 메뉴 자동 변경)
 ┃ ┃ ┣ 📂 properties/      # (공통) 매물 관리 (일반인: 위탁현황 / 중개사: CRM)
 ┃ ┃ ┣ 📂 messages/        # (공통) 1:1 실시간 쪽지/상담 내역
 ┃ ┃ ┣ 📂 articles/        # (공통) 뉴스 기사 초안 작성 및 제출 구역
 ┃ ┃ ┣ 📂 studies/         # (공통) 공실스터디(교육/강의) 개설 신청
 ┃ ┃ ┣ 📂 boards-manage/   # (중개사 전용) 자기 부동산 서브도메인 전용 게시판 관리
 ┃ ┃ ┣ 📂 tenant-config/   # 🎨 (중개사 전용) 서브도메인 테마(스킨) 강제 변경 및 직원 승인
 ┃ ┃ ┗ 📂 super-admin/     # 👑 (본사 전용) 최고관리자 전용 통제 구역
 ┃ ┃     ┣ 📂 theme-maker/ # 🎨 최고관리자 전용 홈페이지 테마(스킨) 템플릿 생성 팩토리
 ┃ ┃     ┣ 📂 approvals/   # 뉴스 기사 데스킹 및 스터디(강의) 개설 최종 승인 통제소
 ┃ ┃     ┣ 📂 tenants/     # 가맹 부동산 서브도메인 및 트래픽 총괄 관리
 ┃ ┃     ┗ 📂 global-boards/# 본사 직속 공용 게시판(자료실) 개설 및 권한 제어
 ┃ ┃
 ┃ ┗ 📂 [tenant]/          # 🏢 B2B 부동산 서브도메인 프론트엔드 홈 (예: happy.gongsilnews.com)
 ┃   ┣ 📜 layout.tsx       # 🎨 접속된 서브도메인의 DB 설정값을 읽어 테마(스킨)를 즉각 씌워주는 뼈대
 ┃   ┣ 📜 page.tsx         # 개별 부동산 고객 노출용 홈 화면
 ┃   ┗ 📂 boards/          # 소장님이 직접 만든 자기 부동산 전용 게시판 화면
 ┃
 ┣ 📂 components/          # 2️⃣ 레고 블록(UI 부품) 보관소
 ┃ ┣ 📂 layout/            # 사이트 전체 뼈대 부품 (Header, Footer)
 ┃ ┣ 📂 widgets/           # 기능 위젯 (NewsCard, BoardList, PropertyCard)
 ┃ ┣ 📂 admin/         # 관리자 전용 컴포넌트 (Sidebar, DataTable)
 ┃ ┗ 📂 ui/                # 아토믹(Atomic) 디자인 부품 (Button, Input)
 ┃
 ┣ 📂 lib/                 # 3️⃣ 백엔드 엔진 & 서버 로직 보관소
 ┃ ┣ 📜 supabase.ts        # Supabase 연동 코드 및 접속 키
 ┃ ┗ 📜 fetchers.ts        # DB 통신 함수 모음
```

## 🎨 다이내믹 테마 스킨(Theme) 엔진 아키텍처

공실뉴스의 서브도메인 시스템은 단순히 로고나 색상 하나를 바꾸는 것을 넘어, **'네이버 블로그 스킨'**처럼 구조와 분위기 자체가 완전히 바뀌는 강력한 **다이내믹 테마 엔진**을 탑재합니다.

* **최고관리자 (테마 공급자)**
  * `dashboard/super-admin/theme-maker` 공간에서 본사 디자이너/관리자가 미리 **[모던 화이트], [클래식 네이비], [보테니컬 그린], [하이테크 다크]** 등 고도화된 테마 템플릿을 개발하여 '테마 스토어 목록'에 등록합니다.
  
* **가맹 부동산 소장님 (테마 소비자)**
  * 자사 관리자 화면(`dashboard/tenant-config`)의 내 홈페이지 꾸미기 메뉴에서, 본사가 만들어둔 '테마 목록' 중 하나를 입맛에 맞게 고르고 메인 배너 슬라이더용 사진만 등록하면 **소장님의 홈페이지(서브도메인) 전체 레이아웃과 컬러가 0.1초 만에 즉각 변신**합니다.

* **프론트엔드 연동 방식 (Next.js & Tailwind CSS Variables)**
  * 브라우저가 접속할 때 `[tenant]/layout.tsx` 가 동작하면서 먼저 DB에서 해당 부동산이 고른 테마 ID를 읽어옵니다.
  * 그 즉시 최상단 HTML의 CSS Variable(변수) 값을 치환하여, 하위의 모든 Tailwind 컴포넌트들이 약속된 테마의 색상과 여백으로 렌더링되게 만듭니다. (자바스크립트 리렌더링 버벅임 없이 Server Side에서 즉시 주입되어 초고속 체감 속도를 보장합니다.)

## 👑 본사(SUPER_ADMIN) 패널과 권한 분리

통합 접속창구라도 최고관리자(`super-admin/`) 영역은 RLS 데이터 분리로 차원이 다른 통제권을 가집니다.

1. **승인 게이트키퍼 권한**: 기사 초안, 스터디 개설안은 최고관리자가 [데스킹/승인]을 거쳐야만 세상에 노출됩니다.
2. **글로벌 통계 및 생사여탈권**: 전국 11만 부동산의 수익, 트래픽을 관제하고 블랙리스트 지점을 즉시 차단합니다.
3. **자금 및 수익 정산**: 전역 결제(PG), 배너, 강사 수수료의 현금 흐름을 총괄합니다.

## 📌 다이내믹 자료실 / 게시판 권한 생성 아키텍처

'고정된 게시판'이 아니라 필요에 따라 메뉴를 마음대로 파생시킬 수 있는 시스템을 구축합니다.

1. 본사는 글로벌 자료실/게시판을 무한 생성하고 각 뼈대마다 RLS 권한을 설정합니다.
2. 지점 부동산은 배정된 서브도메인 공간 내에서 독립적으로 자기만의 공지사항/Q&A 게시판을 신설, 삭제, 은닉할 수 있습니다.
