# 공실Talk 기획서

**문서 버전:** 1.0  
**작성일:** 2026-04-24  

---

## 1. 개요
기존 '댓글·문의 관리'를 **'공실Talk'**이라는 소통 플랫폼으로 리브랜딩합니다.
단순한 댓글 확인 도구가 아닌, 회원 간 실시간 대화를 지원하는 커뮤니케이션 허브로 발전시킵니다.

## 2. UI/UX 구성

### 좌측: 대화 목록 (채팅방 리스트)
- 탭: `전체` / `공실 매물` / `뉴스 기사`
- 각 항목은 카카오톡 채팅방 목록 스타일로 표시
  - `[출처 배지]` + `[상대방 이름]` + `[마지막 메시지 미리보기]` + `[시간]` + `[안 읽은 개수]`
- 검색창: 작성자, 내용, 관련 글 제목 검색

### 우측: Talk 패널 (채팅 대화창)
- **상단 헤더**: 출처 정보 (매물/기사 제목)  + 닫기 버튼
- **대화 영역**: 카카오톡 말풍선 스타일
  - 상대방(손님/회원) 말풍선 → 좌측, 프로필 아이콘 + 이름 + 시간
  - 나(관리자/부동산 사장님) 말풍선 → 우측, 파란색 배경
  - 시간순 정렬 (위→아래, 최신이 제일 밑)
- **하단 입력창**: 텍스트 입력 + 전송 버튼 + 비밀글 체크박스

## 3. 핵심 차별점
- 기존: 댓글 → 답변 (1회성 게시판 느낌)
- 공실Talk: 연속적인 대화 흐름 (메신저/채팅 형태), 회원 간 실시간 소통

## 4. 확장 가능성
- 일반 회원용 공실Talk: 매물 상세 페이지에서 부동산에 바로 질문
- 알림 연동: 새 Talk 시 🔔알림, 향후 카카오 알림톡 연계
- AI 에이전트 연동: 자동 응답 및 FAQ 처리
