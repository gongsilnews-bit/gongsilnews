### 1. 관리자 메뉴(사이드바) UI 수정
- 관리자(Super Admin) 페이지의 왼쪽 메뉴 중 `[플러그인]` 메뉴명을 `[부동산홈페이지]`로 변경

- 부동산 관리자에서 `홈페이지` 메뉴 클릭 시 "준비 중인 기능입니다" 알림 표시 적용 완료

### 2. 회원가입/로그인 모달 마케팅 최적화 (`AuthModal.tsx`)
- 기존 로고 이미지를 선명한 SVG 코드로 교체하고 사이즈를 확대 (64x64)
- 로고 중앙 재생 버튼(노란색 원 내부)을 정삼각형 형태의 수학적 수치로 재조정
- 모달 내 혜택 설명 문구를 임팩트 있는 카피로 전면 수정:
  - "11만 부동산 무료 정보채널"
  - 로컬 부동산이 직접 전달하는 시세 현황 뉴스
  - 중개사에서 꼭 필요한 AI 유튜브 특강 시청
  - 대한민국 부동산 누구나 가입하는 100% 무료 공동중개망

### 3. 드론 영상 재생 아이콘 디자인 복구
- 메인 페이지의 프리미엄 드론 영상 목록(`PremiumDroneCarousel.tsx`) 썸네일 위의 플레이 버튼 디자인을 반투명 흰색 아이콘으로 복구 완료 (`filter-drop-shadow` 적용)

### 4. 중개업소 무료 가입 랜딩 페이지 신설 (`/signup`)
- 직방 CEO 가입 안내 페이지를 모티브로 한 신규 부동산 가입 안내 페이지 신설 (`src/app/signup/page.tsx`)
- 전체 화면 히어로 섹션, 혜택 안내 섹션 (체크리스트 기반 디자인), 3대 핵심 서비스, 추천 타겟 섹션 등 모던한 마케팅 랜딩 페이지 구현
- 헤더의 `[중개업소무료가입]` 메뉴 클릭 시 신규 랜딩 페이지로 렌더링되도록 라우터 연결 업데이트

### 5. 하단(Footer) 외부 기관 링킹 슬라이더 배너 추가
- `src/components/RealEstateInfoBanner.tsx` 신규 컴포넌트 개발
- 여기와방 사이트의 하단 배너를 참고하여, 총 12개의 정부 및 부동산 기관 로고(국토교통부, 인터넷등기소, 일사편리 등)를 세련된 형태의 버튼 배너로 디자인
- 전체 가로 사이즈를 1200px 중앙 정렬에 맞추었으며, 한 화면에 6개씩 정렬
- **수동 슬라이더 기법** 도입: 좌측(`<`)/우측(`>`) 화살표 버튼을 클릭하면 한 페이지만큼 슬라이딩으로 전환되도록 구현

### 10. 헤더 하위 메뉴 변경
