# 2026-05-18 개발 일지: 모바일 UX 및 지도 줌 최적화

이 문서는 2026년 5월 18일 진행된 공실뉴스 플랫폼의 모바일 편의성(UX) 및 카카오맵 관련 버그 개선 사항을 기록합니다.

---

## 1. 모바일 뉴스 레이아웃 겹침 버그 수정
- **수정 파일:** `src/app/m/news/MobileNewsClient.tsx`
- **주요 내용:** 모바일 환경에서 스크롤 시 상단 카테고리 탭이 고정 헤더 뒤로 숨겨지는 문제 발생. 컨테이너 레이아웃을 `position: fixed`로 고정하고 내부 리스트 영역만 `overflow-y: auto`로 스크롤되도록 구조를 변경하여, 스크롤을 내려도 상단 탭이 항상 안정적으로 고정되도록 수정했습니다.

## 2. 모바일 기사 상세뷰 에디터용 삭제 버튼(X) 노출 방지
- **수정 파일:** `src/app/m/news/MobileNewsClient.tsx`
- **주요 내용:** 관리자용 에디터(Quill)에서 이미지 첨부 시 자동으로 삽입되는 관리자용 투명 삭제 버튼(`<button class="editor-media-delete">`)이 모바일 '우리동네뉴스' 슬라이드 뷰 기사 본문에 그대로 렌더링되는 현상을 수정했습니다. 기사 본문을 렌더링할 때 정규식(Regex)을 사용해 해당 `button` 태그를 필터링하여 말끔하게 제거했습니다.

## 3. 우리동네뉴스 모바일/PC 지도 줌(Zoom) 최적화
- **수정 파일:** 
  - `src/app/m/news/MobileNewsClient.tsx`
  - `src/app/(map)/news_map/NewsMapClient.tsx`
- **주요 내용:** 
  - **초기 줌 레벨 조정:** 모바일 진입 시 지도가 너무 넓은 한반도 전역(레벨 7)을 비춰 조작이 불편했던 점을 반영하여, 시/군 단위가 적절히 보이는 레벨 6으로 초기값을 조정했습니다.
  - **클러스터 클릭 시 과도한 줌인 방지:** 같은 주소의 기사가 여러 개 겹쳐 묶인 클러스터(동그란 숫자 마커)를 클릭할 때, 카카오맵이 해당 지점들의 여백을 계산하다 무한히 확대하여 텅 빈 하얀 배경(레벨 1~2)만 띄워버리는 버그를 수정했습니다.
  - **커스텀 줌 로직 적용:** 클러스터 기본 확대 동작을 끄고(`disableClickZoom: true`), 클릭 시 부드럽게 2단계씩만 확대하되 너무 좁은 영역(레벨 3 미만)으로는 더 이상 확대되지 않도록 제한하는 로직을 새로 추가했습니다.

## 4. 모바일 공실등록 전화번호 자동 하이픈(-) 포맷팅 적용
- **수정 파일:** `src/app/m/admin/vacancy/write/page.tsx`
- **주요 내용:** 모바일 기기에서의 입력 편의성을 극대화하기 위해 숫자만 입력해도 자동으로 전화번호 형식에 맞춰 하이픈이 삽입되는 `formatPhone` 유틸리티 함수를 추가했습니다.
  - **적용 대상 필드:** 임대인 연락처, 등록자 연락처, 부동산 일반전화, 부동산 휴대번호
  - 서울 지역번호(02), 일반 휴대폰 번호(010), 전국 대표번호(1588 등) 패턴을 모두 인식하여 실시간으로 정확한 위치에 `-`를 넣어줍니다.
