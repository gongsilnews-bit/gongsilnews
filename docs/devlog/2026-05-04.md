# 개발 일지 (2026-05-04) - 중개보수 및 금액 표기 표준화 작업

## 작업 개요
플랫폼 내의 모든 공실광고 금액 표기 방식을 "억 천" 단위로 표준화하고, 중개보수 배지(아이콘)가 누락된 곳을 보완하며 권한에 따른 노출 로직을 일원화함.

## 주요 작업 내용

### 1. 금액 표기 방식 표준화 ("억 천" 로직)
- **목표**: `150,000,000`원과 같은 금액을 사용자가 직관적으로 읽을 수 있도록 `1억 5천만` 형식으로 변환하여 표시.
- **적용 대상**:
  - PC/모바일 리포터 프로필 (`PCReporterClient.tsx`, `MobileReporterClient.tsx`)
  - 지도 기반 공실 리스트 (`GongsilClient.tsx`, `MobileGongsil page.tsx`)
  - 메인 페이지 및 히어로 맵 섹션 (`HeroMapSection.tsx`, `Homepage page.tsx`)
  - 뉴스 상세 및 카드 컴포넌트 (`NewsReadContent.tsx`, `MobileNewsClient.tsx`, `RealtorPropertyCard.tsx`)
- **수정 로직**: `formatAmount` 또는 `formatPrice` 함수 내에서 만 단위 절삭 후 `억`과 `천` 단위를 명시하고, 마지막에 `만` 단위를 붙여 가독성 향상.

### 2. 중개보수 배지(아이콘) 표준화 및 보완
- **목표**: 중개보수 정보를 권한(`userLevel >= 2`)이 있는 사용자에게만 노출하고, 누락된 화면에 배지를 추가.
- **신규 추가 및 수정**:
  - **모바일 뉴스 (`MobileNewsClient.tsx`)**: 공실 리스트 아이템에 중개보수 배지 추가.
  - **부동산 매물 카드 (`RealtorPropertyCard.tsx`)**: 상세 보기 및 미니 리스트 아이템에 중개보수 배지 추가.
- **노출 로직**:
  - `userLevel >= 2` (정회원 이상)인 경우에만 렌더링.
  - 하드코딩된 "법정수수료" 대신 데이터베이스의 `realtor_commission` 또는 `commission_type` 값을 우선적으로 표시.
- **디자인**: 빨간색 테두리와 굵은 글씨(`#fa5252`)의 배지 스타일로 통일.

### 3. 소스 제어 및 배포 준비
- 모든 변경 사항을 검토하고 `git`을 통해 커밋 및 푸시 완료.
- **커밋 메시지**: `Standardize price formatting to '억 천' logic and add missing commission badges`
- **변경 파일**: 총 9개 파일 수정.

- 사용자 레벨에 따른 데이터 마스킹 로직이 `RealtorPropertyCard`와 같은 공용 컴포넌트에서 일관되게 작동하는지 지속적인 모니터링 필요.
