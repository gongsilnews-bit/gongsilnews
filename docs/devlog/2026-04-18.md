### 4. SSR(서버 사이드 렌더링) 성능 최적화
- **문제**: 배너 영역과 부동산 특강이 페이지 로드 후 지연 로딩(깜빡임 현상)
- **원인**: 클라이언트 사이드에서 비동기 데이터 요청 방식 (`useEffect` → `getBannersByPlacement`)
- **해결**: 서버 컴포넌트에서 `Promise.all`로 데이터를 미리 가져와 `initialBanners`/`initialLectures` prop으로 전달
- **적용 대상 컴포넌트**:
  - `BannerSlot.tsx` — `initialBanners` prop 추가
  - `HeaderTextBanner.tsx` — `initialBanners` prop 추가
  - `SpecialLectureBanner.tsx` — `initialLectures` prop 추가
- **적용 대상 페이지**:
  - `(main)/layout.tsx` — `TOP_FULL`, `HEADER_TEXT` 배너 SSR 주입
  - `(main)/page.tsx` — `MAIN_TOP`, `MAIN_BOTTOM_FULL` 배너 + 특강 데이터 SSR 주입
  - `CategoryNewsGrid.tsx` — `MAIN_ISSUE_RIGHT` 우측 배너 SSR 주입
