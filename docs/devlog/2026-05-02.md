# 개발 일지 - 2026년 4월 29일 ~ 5월 2일 (최근 3일간 작업 내역)

최근 3일 동안 진행된 작업 내역을 주요 도메인 및 기능별로 중복 없이 정리한 내용입니다.

## 1. 모바일 공실 열람 (Vacancy Search) 개선
* **상세 보기 화면 고도화**:
  * 연락하기 CTA(Call To Action) 버튼 텍스트 변경 및 실제 전화 연결 기능 고도화
  * 하단 고정 레이아웃(Sticky Bottom) 안정성 강화
  * 매물 번호 표시를 UUID에서 실제 식별 번호(`vacancy_no`)로 변경
  * 공실 상세 진입 시 스크롤 최상단 초기화 보장 및 모바일 하드웨어 뒤로가기 버튼 호환성 개선
  * 등록자 프로필 이미지 깨짐 방지 및 에러 시 기본 플레이스홀더 처리
  * 첨부된 매물 사진이 없을 경우 비어 있는 기본 썸네일 영역 숨김 처리
* **지도 및 마커 UI 개선**:
  * 공실 열람 지도 마커 및 버튼 색상을 오렌지 톤에서 네이비(`#1a2e50`)로 변경하여 시인성 확보
  * 마커 클릭 이벤트 버블링 문제를 수정하여 모바일 바텀시트(Bottom Sheet)가 정상 작동하도록 안정화
  * 홈 화면 미니 공실 지도 마커에 단순 '공실' 텍스트 대신 실제 **해당 지역 매물 갯수**를 표시하도록 기능 추가

## 2. 모바일 뉴스 및 전역 UI/UX 디자인 개편
* **모바일 네비게이션 및 메뉴 고도화 (네이버 스타일 적용)**:
  * 모바일 홈/뉴스 화면 상단 카테고리 탭에 좌우 스와이프 슬라이드 애니메이션 및 네이버 스타일 디자인 전면 도입
  * 메뉴 폰트 크기, 간격, 선택된 탭의 하단 분리선 및 색상 테마 정밀 조정 (가독성 및 사용성 최적화)
  * 홈 화면과 모바일 헤더 간의 불필요한 공백을 제거하기 위해 헤더를 고정(Fixed) 속성으로 변경 및 패딩 최적화
  * 햄버거 메뉴 닫기 시 발생하는 고스트 클릭(Ghost Click) 이슈 방지 처리
  * 모바일 전체 클릭 먹통 현상을 해결하기 위해 `GongsilTalkOverlay` 컴포넌트의 `pointerEvents` 속성 수정
* **뉴스 상세 및 리스트 레이아웃 개선**:
  * 모바일 뉴스 리스트 컴포넌트 서버 마이그레이션(초기 로딩 지연 제거)
  * 기사 목록 터치 시 뚜렷한 회색(`#f3f4f6`) 시각적 피드백 효과 적용 및 불필요한 텍스트 블록 선택 방지
  * 전체 폰트 크기를 기존 대비 2px 상향 조정하여 모바일 가독성 증대
  * '우리동네부동산' 등 뉴스 썸네일 크기를 확대하고, 2개가 자연스럽게 노출되도록 조정
* **이미지 렌더링 최적화**:
  * 전역 로고 이미지를 새로운 로고(`logo.png`, `logo_main.png`)로 전면 교체
  * 브라우저 캐싱 문제를 해결하기 위해 쿼리 파라미터 캐시 무효화 적용
  * `Next.js Image` 컴포넌트 도입을 통한 정적 이미지 모바일 최적화 적용

## 3. 모바일 기사 관리자 도구 (Admin / Editor)
* **기사 작성 폼 편의성 개선**:
  * 1차/2차 카테고리 연동 선택 시스템을 PC 관리자 환경과 동일하게 구현하여 다이나믹한 셀렉트 박스 적용
  * 부제목(Subtitle) 입력란을 초기부터 항상 노출되도록 레이아웃 수정
  * **통합 미디어 UI 구축**: 사진과 유튜브 영상을 동일한 공간에서 다중 첨부할 수 있도록 UI 개편
  * 첨부된 사진과 영상 중 원하는 항목을 직관적으로 **'대표(Cover)' 이미지/영상으로 지정**하는 기능 추가 및 저장 연동

## 4. 공실톡 (GongsilTalk 메신저) 고도화
* 모바일 환경 진입 시 우측에서 화면 전체로 부드럽게 등장하는 **슬라이드 인(Slide-in) 애니메이션** 적용
* 모바일 채팅 화면 상단에 명시적인 뒤로가기 버튼 추가 (PC는 기존의 분할 화면 방식 유지)
* 매물 정보 공유(Property Sharing) UX 개선 및 등록자 프로필 표시 버그 수정
* 하단 플로팅 버튼(Floating Button) 위치를 화면 및 상태바 환경에 맞추어 재조정
