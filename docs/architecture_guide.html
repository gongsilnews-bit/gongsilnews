⚡핵심 규칙

1. 관리자 UI → gongsilnews에서 개발
부동산 사장님이 설정하는 모든 화면은 gongsilnews 프로젝트 (src/app/realty_admin/)
2. 고객 화면 → homepage에서 개발
*.gongsilnews.com 방문자가 보는 모든 화면은 homepage 프로젝트
3. Supabase = 중간 다리
gongsilnews가 쓰고, homepage가 읽음. 스키마 변경 시 양쪽 확인 필요
4. 기사는 새로 작성할 필요 없음
공실뉴스에서 이미 작성한 기사를 member_id로 필터해서 homepage에 자동 표시
