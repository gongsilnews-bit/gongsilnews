# 모바일 기사 개인화 & 카테고리별 맞춤형 타이틀 설계 (Docs)

본 문서는 공실뉴스 모바일 플랫폼의 각 뉴스 카테고리(1차 및 2차 카테고리) 진입 시, 상단 2차 카테고리 바로 아래에 노출될 **"개인화 맞춤 타이틀"**에 대한 기획서 및 명세서입니다.

---

## 1. 기본 공식 및 설계 원칙

### 1.1 기본 구조
```text
[회원명 또는 기본명] [호칭/접사]을 위한 [맞춤형 카테고리 멘트] News
```

### 1.2 사용자 상태(로그인 여부) 대응
- **로그인 완료 상태**: 
  - 사용자의 실제 이름(`memberName`)을 데이터베이스(members 테이블)에서 가져와 바인딩합니다.
  - 예시: `김민종 대표님을 위한 ...` / `이지은 대표님을 위한 ...`
- **미로그인 상태 (비회원)**:
  - 기본 비회원 호칭인 **"부동산 대표님"** 또는 카테고리에 적합한 공통 호칭을 사용합니다.
  - 예시: `부동산 대표님을 위한 ...`

---

## 2. 카테고리별 개인화 타이틀 & 멘트 정의

각 1차 카테고리 및 그 아래의 2차 카테고리(칼럼)별로 최적화된 멘트를 설계하여 정보의 소구력을 극대화합니다.

### 2.1 공실뉴스 (`news_gongsil`)
> **컨셉**: 실무적이고 실시간 가치가 높은 정보 전달

| 2차 카테고리 (칼럼) | 추천 맞춤형 타이틀 공식 | 예시 (회원: 김민종, 비회원 기준) |
| :--- | :--- | :--- |
| **전체 (Default)** | `{이름} 대표님을 위한 실시간 공실 소식` | `김민종 대표님을 위한 실시간 공실 소식 News` |
| **아파트/오피스텔** | `{이름} 대표님을 위한 아파트·오피스텔 맞춤 정보` | `김민종 대표님을 위한 아파트·오피스텔 맞춤 정보 News` |
| **빌라/주택** | `{이름} 대표님을 위한 빌라·주택 핵심 매물 뉴스` | `부동산 대표님을 위한 빌라·주택 핵심 매물 뉴스 News` |
| **원룸/투룸(풀옵션)**| `{이름} 대표님을 위한 원룸·투룸 실무 트렌드` | `김민종 대표님을 위한 원룸·투룸 실무 트렌드 News` |
| **상가/사무실/공장/토지**| `{이름} 대표님을 위한 상업용·토지 자산 정보` | `김민종 대표님을 위한 상업용·토지 자산 정보 News` |
| **신축/분양/경매** | `{이름} 대표님을 위한 신축·분양·경매 알짜 정보` | `부동산 대표님을 위한 신축·분양·경매 알짜 정보 News` |

## 3. 기술적 구현 방안 (Next.js / React)

### 3.1 상태 의존성 데이터 매핑 구조
타이틀 렌더링 시 현재 활성화된 1차 카테고리(`activeTab`)와 2차 카테고리(`section2Tab`)의 상태값을 기반으로 문구를 매핑하는 객체를 생성하여 구현합니다.

```typescript
// 1차 카테고리 & 2차 카테고리별 문구 맵 정의
const PERSONALIZED_MENTAL_MAP: Record<string, Record<string, string>> = {
  "news_gongsil": {
    "전체": "실시간 공실 소식",
    "아파트/오피스텔": "아파트·오피스텔 맞춤 정보",
    "빌라/주택": "빌라·주택 핵심 매물 뉴스",
    "원룸/투룸(풀옵션)": "원룸·투룸 실무 트렌드",
    "상가/사무실/공장/토지": "상업용·토지 자산 정보",
    "신축/분양/경매": "신축·분양·경매 알짜 정보"
  },
  "news_politics": {
    "전체": "부동산·경제 맞춤 브리핑",
    "부동산 정책/동향": "급변하는 부동산 정책/동향",
    "경제/재테크/주식": "거시경제·재테크 바이블",

```

### 3.2 렌더링 예시 컴포넌트 마크업
```tsx
const userName = currentUser?.memberData?.name || "부동산";
const activeSubCategory = section2Tab || "전체";
const mentalText = PERSONALIZED_MENTAL_MAP[activeTab]?.[activeSubCategory] || "추천 뉴스";

return (
  <div style={{ padding: "16px 16px 4px", backgroundColor: "#fff" }}>
    <h2 style={{ fontSize: "16px", fontWeight: "normal", color: "#333", margin: 0, letterSpacing: "-0.5px" }}>
      <span style={{ fontWeight: "800", borderBottom: "2px solid #333", paddingBottom: "2px" }}>
        {userName} 대표님
      </span>
      <span>을 위한 </span>
      <span style={{ fontWeight: "800", color: "#ea580c" }}>{mentalText} News</span>
    </h2>
  </div>
);
```

이 기획안을 바탕으로 회의를 거친 뒤 디테일한 글꼴 크기, 여백, 선 등을 조율하고 실제 코드에 반영하도록 하겠습니다.
