공실등록 폼 (Vacancy Registration Form) DB & UI 매핑 설계도 (최종 완전판)

작성일: 2026-06-04
목적: 장애 발생 시 완전한 프론트/백엔드 데이터 복구 및 아키텍처 파악을 위한 1:1 DB 구조 바이블.

7. 실제 UI 입력 컴포넌트 세부 명세 (프론트엔드 복구용)

각 항목별로 실제 화면(DOM)에서 렌더링되는 입력 양식(Input Type)과 안내 문구(Placeholder)를 상세히 기록합니다. 프론트엔드 UI 복구 시 픽셀 단위로 동일한 경험을 제공하기 위한 목적입니다.
