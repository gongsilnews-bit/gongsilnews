# 공실등록 폼 QA 테스트 리포트 (아파트 편)

> **[재부팅 후 봇 실행 대기 지시서 - User Requirements]**
> 1. **로그인**: 브라우저를 열고 `gongsilmarketing@gmail.com` 으로 자동 로그인 (비밀번호 모를 시 디폴트 비밀번호 시도 또는 우회)
> 2. **사용자 허락 생략**: 대표님의 허락을 매번 구하지 않고 봇이 알아서 끝까지 다 입력하고 제출할 것.
> 3. **입력 조건 (중요)**:
>    - 대분류: 아파트·오피스텔 / 소분류: 아파트
>    - 거래유형별로 각각 **2건씩** 등록 (매매 2, 전세 2, 월세 2, 단기 2 = 총 8건)
>    - **사진**: 로컬 업로드 대신 화면의 **"포토DB"** 버튼을 눌러서 기존 DB 사진 임의 선택
>    - **AI 초안 글쓰기 절대 금지!**: 버튼 누르지 말고 텍스트 영역에 직접 가짜 텍스트 타이핑할 것
> 4. **이슈 기록**: 봇이 입력하다 막히거나, UI에 문제가 있으면 코드를 절대 수정하지 말고 이 문서 하단에 에러 현상만 꼼꼼히 기록할 것.
